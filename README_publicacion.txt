Publicación gratis con GitHub Pages

1. Crea un repositorio nuevo para la web, por ejemplo: recetas-hellofresh-web
2. Sube el contenido de esta carpeta tal cual.
3. En GitHub: Settings > Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Guarda. Tu web quedará publicada en unos minutos.
